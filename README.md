# Crypto Sandbox (React)

An interactive AES-256 crypto sandbox designed for students and developers to experiment with secret keys, encryption, and decryption directly in the browser.

This small tool allows you to safely explore how symmetric encryption works using AES with a simple interface built in React.

---

## Purpose

The goal of this project is to provide a simple sandbox environment where students can experiment with:

- encryption
- secret keys
- ciphertext
- browser-side cryptography

---

## How it works

The application uses CryptoJS AES to encrypt and decrypt messages.

### Encryption

1. Enter a message  
2. Enter a secret key  
3. The encrypted hash is generated automatically  

### Decryption

1. Paste the encrypted hash  
2. Enter the same key used for encryption  
3. The original message will be revealed  

If the key is incorrect or the hash is invalid, the decryption will fail.
